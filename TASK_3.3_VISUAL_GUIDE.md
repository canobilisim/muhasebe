## State Persistence

Note: `connectionStatus` is **NOT** persisted to localStorage. It's always initialized as `'disconnected'` on app start and updated based on actual connection checks.

Persisted fields:
- user
- profile
- session
- isInitialized
- isAuthenticated
- userRole
- branchId

Non-persisted (runtime only):
- connectionStatus â† Always starts fresh
- lastError
- isLoading
- error
- initializeAttempts
- debugMode
